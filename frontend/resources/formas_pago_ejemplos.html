Carga únicamente los métodos de pago que están activos.

3. Obtener un Método Específico

Carga un método de pago por su ID.

4. Crear Nuevo Método de Pago

Ejemplo de cómo agregar un nuevo método programáticamente.

5. Usar el JavaScript Manager

Ejemplos de código para usar desde JavaScript:

Ejemplo 1: Cargar todos los métodos

const methods = await paymentManager.fetchAllPaymentMethods(); console.log('Métodos disponibles:', methods);

Ejemplo 2: Obtener solo activos

const active = await paymentManager.fetchActivePaymentMethods(); active.forEach(method => { console.log(`${method.name} - ${method.reference}`); });

Ejemplo 3: Crear método

const newMethod = await paymentManager.createPaymentMethod({ type: 'tarjeta', name: 'Mi Tarjeta', reference: 'CARD-NEW', description: 'Nueva forma de pago', instructions: 'Pasos para pagar', status: 'activo' });

Ejemplo 4: Actualizar método

const updated = await paymentManager.updatePaymentMethod(1, { name: 'Tarjeta Actualizada', status: 'inactivo' });

Ejemplo 5: Eliminar método

const result = await paymentManager.deletePaymentMethod(1); if (result.success) { console.log('Método eliminado'); }

Ejemplo 6: Validar datos

const validation = paymentManager.validatePaymentData({ type: 'tarjeta', name: 'Mi Método', reference: 'REF-001' }); if (validation.valid) { console.log('Datos válidos'); } else { console.log('Errores:', validation.errors); }

6. Información Técnica

Estructura de datos de un método:

{ "id": 1, "type": "tarjeta", "name": "Tarjeta de Crédito/Débito", "reference": "CARD-001", "description": "Realiza pagos seguros", "instructions": "Ingresa los datos", "status": "activo", "created_at": "2024-05-20 15:07:17", "updated_at": "2024-05-20 15:07:17" }
