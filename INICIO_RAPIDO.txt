# 🎯 INICIO RÁPIDO - STEL BOLIVIA WEB

## 🚀 Para Empezar Ahora Mismo

### 1️⃣ Ver el Sitio
```
Opción A: Abre en navegador → frontend/index.html
Opción B: Si tienes PHP habilitado → http://localhost/stelcompany/frontend/index.html
```

### 2️⃣ Ver Página de Comunicados
```
frontend/resources/comunicados.php
→ Haz click en cualquier comunicado
→ Se abre MODAL
→ Haz click en X para cerrar
```

### 3️⃣ Crear Nueva Página
```php
<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="../assets/styles.css">
</head>
<body>
    <?php include '../layouts/header.php'; ?>
    
    <main>
        <section class="section">
            <div class="container">
                <!-- Tu contenido -->
            </div>
        </section>
    </main>
    
    <?php include '../layouts/footer.php'; ?>
    <script src="../assets/script.js"></script>
</body>
</html>
```

## 🎨 Componentes Listos para Usar

### Botones
```html
<button class="btn btn-primary">Primario</button>
<button class="btn btn-secondary">Secundario</button>
<button class="btn btn-ghost">Ghost</button>
```

### Tarjetas
```html
<div class="card">
    <h3>Título</h3>
    <p>Contenido</p>
</div>
```

### Modales
```html
<div class="modal" id="miModal">
    <div class="modal-content">
        <button class="modal-close">&times;</button>
        Contenido
    </div>
</div>
```

### Grid Responsivo
```html
<div style="display: grid; grid-template-columns: repeat(auto-fit, minmax(280px, 1fr)); gap: 30px;">
    <div class="card">Item</div>
</div>
```

## 📞 Preguntas Frecuentes

**P: ¿Cómo cambio colores?**  
R: Edita variables en `frontend/assets/styles.css` (líneas 7-13)

**P: ¿Cómo cambio el logo?**  
R: Edita SVG en `frontend/layouts/header.php` (líneas 10-30)

**P: ¿Cómo agrego una página nueva?**  
R: Copia el template de arriba a `frontend/resources/mi-pagina.php`

**P: ¿Dónde agrego Supabase?**  
R: Backend config + actualizar `frontend/assets/script.js` con fetch real

**P: ¿Es responsive?**  
R: Sí, probado en mobile, tablet, desktop
